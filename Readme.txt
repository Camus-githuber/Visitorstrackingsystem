
Welcome to the Bortech visitors tracking system

 how to install the project

1 Open phpmyadmin on localserver[XAMPP, WAMP, LAMP or any]

2 create a database named  'bortech_visitors_record'

3 import the database from the folder named  bortech_visitors_record.sql

4 Copy the folder 'Visitors_Tracker_System' and paste it in your server folder[XAMPP = htdocs, WAMP=www]

5 

5 open chrome or your preferrred browser and place the cursor on the URL tab and paste 'localhost/Visitors_Tracker_System'
